静态账号管理

生成和维护 accounts.json，不连接 Neon / Node 后端。适合临时 Pro 账号、早期售卖和手动发放。

重要：accounts.json 只保存哈希；「明文账号清单」才是发给用户的用户名和密码。 如果你的 GitHub 仓库是公开的，不要把明文账号清单提交到仓库。用户登录时只需要 index.html、app.html 和 accounts.json。
