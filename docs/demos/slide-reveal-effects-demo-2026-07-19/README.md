# 幻灯片笔迹效果 Demo 归档

## 当前选择

以下三种效果已正式接入 `whiteboard.html` 和 `whiteboard-pro.html`，本目录继续作为视觉与算法参考：

1. 彩铅铺色（效果 1，建议默认）
2. 水墨晕染（效果 2）
3. 左上 → 右下铅笔绘制（效果 5；正式 UI 可简称“铅笔描绘”）

效果 3“逐笔描绘”和效果 4“左 → 右铅笔绘制”未接入正式版本，但保留源码供以后参考。

正式版本另外恢复了五效果 Demo 之前已有的旧版直线斜切笔迹，界面命名为“斜线推进”。它不是本目录中的效果 4，而是从 WhiteBoard 的 Git 原实现恢复其直线裁切和虚线前沿；正式版后来已与其他三种效果统一为“墨线起稿 → 彩色上色”的约 4.4 秒两阶段时序。

## 后续正式接入约束

- 同步修改项目根目录的 `whiteboard.html` 和 `whiteboard-pro.html`，不要只接入其中一个版本。
- 保持一个“笔迹”入口；当前正式版本在效果 1、2、5 之外，还提供旧版“斜线推进”作为第四个预设。主按钮播放当前效果，下拉按钮切换效果。
- 默认使用“彩铅铺色”，仅记住用户最后一次选择，不增加速度、方向、浓度等高级控件。
- 真实笔迹绘制必须进入白板 Canvas 和录制成品；模式弹窗、缩略图、按钮与进度提示不得进入录像。
- 正式实现应复用 WhiteBoard 现有幻灯片缓存和 `slideReveal` 渲染链，不要直接整段复制 Demo 的页面 UI。
- 继续使用固定随机种子，保证重播和录制结果稳定；尊重 `prefers-reduced-motion`。
