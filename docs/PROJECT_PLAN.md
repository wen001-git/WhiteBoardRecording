# 白板录制工具 · 项目计划（PROJECT_PLAN）

> 单一产品文档：需求要点 + 功能清单 + 设计决策 + 测试清单 + 里程碑。
> 运行/状态/TODO 见仓库根 [`AGENTS.md`](../AGENTS.md)；使用说明见 [`README.md`](../README.md)。

## 一句话定位
仿 [excalicord.com](https://excalicord.com/) 的**纯前端单 HTML** 白板录制工具：无限白板画画/写字 + 角落摄像头 + 提词器 + 一键录制导出 mp4。**无后端、零依赖**。

## 需求要点
- 在无限白板上自由绘画/写字，可平移、缩放。
- 摄像头画面出现在角落并录进视频；麦克风录音。
- 内置提词器看稿，且**不被录进视频**。
- 一键录制（开始/暂停/继续/停止）+ 计时，导出 **mp4** 保存本地。
- 双击即用风格：单个 HTML 文件，方便保存/分享。
- 目标用户：老师 / 知识博主 / 远程异步沟通者。

## 功能清单
**白板/画布**：F-1 全屏画布 + 无限平移（hand 工具 / 空格拖拽）｜F-2 滚轮以光标为中心缩放 + 比例显示｜F-3 对象数组存储、重绘不丢。
**绘图工具**：F-4 画笔（pointer，支持鼠标/触屏/笔）｜F-5 颜色｜F-6 粗细（细/中/粗）｜F-7 矩形｜F-8 圆/椭圆｜F-9 箭头｜F-10 文字（点击输入、可设字号）｜F-11 橡皮擦（对象级命中删除）｜F-12 撤销/重做｜F-13 清空。
**摄像头/音频**：F-14 角落圆形小窗｜F-15 可拖动换位｜F-16 麦克风｜F-17 设备选择（当前用默认，`populateDevices()` 已预留）。
**提词器**：F-18 可开关浮层｜F-19 编辑稿子｜F-20 字号｜F-21 自动滚动 + 调速/播停/回顶｜F-22 **不进录像**。
**录制/导出**：F-23 开始/暂停/继续/停止｜F-24 计时 + 录制中指示｜F-25 白板+摄像头合成为一个视频｜F-26 合并麦克风音轨｜F-27 导出 mp4（原生优先，否则 webm + ffmpeg.wasm 转码回退）｜F-28 停止后预览 + 下载。
**手绘风格（迭代二）**：F-31 矩形/圆/箭头为手绘外观，每个形状带稳定 `seed`、按 seed 确定性生成线条扰动（每帧重绘不闪、不同形状线条各异）；画布文字用手写字体栈。
**录制比例与背景（迭代二）**：F-32 录制设置弹窗——画面比例预设（16:9 YouTube / 4:3 经典 / 3:4 小红书 / 9:16 抖音 / 1:1）+ 背景壁纸（渐变×7 / 纯色×3 / 无）；F-33 可拖动+四角缩放并锁定比例的取景框界定录制区域；F-34 输出 = 选定比例画布上，渐变背景 + 居中白卡片（留白+圆角+阴影）承载取景框内容 + 摄像头按相对位置映射进卡片。

## 关键设计决策
- **架构**：纯前端单 `index.html`，HTML/CSS/JS 全内联。仅当将来要账号/云存储/分享/服务端转码/协作时才加后端——届时本前端逻辑可复用。
- **安全上下文**：`getUserMedia` 需 https 或 localhost；`file://` 多被拦。运行用 `python3 -m http.server` 或部署到静态托管。
- **绘图模型**：`state.scene[]` 对象数组 + `view{x,y,scale}`，每帧重绘；`undoStack/redoStack` 用快照实现撤销/重做。坐标：世界 = (屏幕 − view.xy) / scale。
- **合成录制**：离屏 `recCanvas`（=视口像素）每帧 `drawImage(白板)` + 角落 `drawImage(摄像头)`（圆形裁剪+镜像，位置跟随小窗）→ `captureStream(30)` + 麦克风音轨 → `MediaRecorder`。**用 `setInterval(1000/30)` 驱动合成**（不用 rAF：页面切后台时 rAF 暂停会导致录制黑屏）。
- **mp4 导出**：`MediaRecorder.isTypeSupported('video/mp4;codecs=avc1...')` 为真则直接录 mp4；否则录 webm，提供「转 mp4」按钮**按需**从 CDN 懒加载 ffmpeg.wasm 转码（仅此一处联网）。
- **提词器隔离**：提词器是普通 DOM 浮层，不属于被 `drawImage` 的白板 canvas，天然不进录像。
- **手绘渲染（自写零依赖）**：`mulberry32(seed)` PRNG + `roughLine/roughRect/roughEllipse/roughArrow`；线条扰动幅度随线宽。形状对象存 `seed`，重绘按 seed 确定性生成→稳定不闪。不引 rough.js，保持单文件。
- **录制比例/背景/取景框**：`RATIOS` 决定输出像素与画布比例；`recConfig{ratio,bgIndex,frame}`；录制状态机 idle→setup→recording→paused。`drawRecFrame` 每帧：填背景渐变 → 画白卡片(留白+圆角+阴影) → 裁剪卡片绘制取景框区域的白板像素 → 摄像头按屏幕相对位置映射进卡片。取景框 `#recFrame` 拖动/缩放时锁定比例。
