# 域名命名建议

## 结论

当前推荐使用：

**record.leewen.work**

它比 `whiteboard.leewen.work` 更适合这个产品，因为用户一眼能理解这是一个“录制”工具，而不是单纯的白板工具。

## 为什么不优先用 whiteboard.leewen.work

`whiteboard` 会让用户以为产品只是画板，不会马上想到“录制短视频”“录制教程”“录屏讲解”。

本项目的核心卖点不是白板本身，而是：

- 录制讲解视频
- 录制教程短视频
- 白板 + 摄像头 + 麦克风 + 屏幕录制
- 导出可发布的视频内容

所以 `record` 更贴近商业定位。

## record.leewen.work 是否适合海外发布

`record.leewen.work` 可以用于 MVP 发布。国外用户大概率不会觉得它不正规，但它不是付费转化最强的域名。

主要原因：

- `.work` 是合法域名后缀，但信任感弱于 `.com` / `.app` / `.ai` / `.io`。
- `leewen` 更像个人品牌，适合独立开发者，但陌生用户第一次看到时不一定知道它代表什么。
- `record.leewen.work` 功能清楚，但整体更像个人工具地址，而不是成熟 SaaS 品牌。

## 推荐策略

短期：

使用 `record.leewen.work` 上线验证，不要因为域名阻塞产品发布。

中长期：

如果产品在 X 或海外渠道验证出真实付费意愿，再购买更产品化的主域名。

可考虑的方向：

- `boardrecord.com`
- `whiteboardrecorder.com`
- `recordboard.app`
- `lessonclip.app`
- `teachclip.app`
- `explainerrecorder.com`

## 产品包装建议

域名可以是 `record.leewen.work`，但产品名不要直接叫域名。

推荐产品名方向：

- LeeWen Record
- BoardRecord
- TeachClip

对外发布时可以这样写：

**LeeWen Record: a whiteboard video recorder for tutorials**  
https://record.leewen.work
